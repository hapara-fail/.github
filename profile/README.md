# hapara.fail

### *Privacy tools and resources for the managed student.*

**[hapara.fail](https://www.hapara.fail)** is an open-source initiative dedicated to exposing and neutralizing the pervasive nature of student surveillance technology. We provide technical insights, practical tools, and a unified blocklist to empower students and advocate for digital privacy in educational environments.
